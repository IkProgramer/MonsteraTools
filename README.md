**MonsteraSteamTool** to zaawansowane narzędzie do testowania gier ze Steam przed podjęciem decyzji o zakupie.  
Wbudowana chmura manifestów, automatyczne zarządzanie biblioteką i pełna sterylność systemu.

[📥 Pobierz najnowszą wersję](https://github.com/IkProgramer/MonsteraTools/releases/download/1.0/MonsteraSteamTool1.0.exe)

</div>

---

## 🎮 Funkcje

| Funkcja | Opis |
|---------|------|
| ☁️ **Baza Monstery** | Wpisz nazwę gry lub GameID – pliki pobiorą się i zainstalują automatycznie |
| 🛡️ **Bezpieczeństwo** | Automatyczna weryfikacja środowiska i ochrona konta |
| 🔄 **Auto-Updater** | Aplikacja sama aktualizuje się do najnowszej wersji |
| ⚡ **Odblokowanie DLC** | Testuj wszystkie dodatki DLC jednym kliknięciem |
| 🔍 **Wyszukiwarka** | Znajdź dowolną grę po nazwie bezpośrednio w aplikacji |
| 🧹 **Czysty System** | Po testach środowisko wraca do pierwotnego stanu |
| 👁️ **Tryb Ukryty** | Ukryj aplikację do zasobnika systemowego jednym kliknięciem |

---

## 📥 Instalacja

1. Pobierz najnowszą wersję z [Releases](https://github.com/IkProgramer/MonsteraTools/releases)
2. Uruchom `MonsteraSteamTool1.0.exe`
3. Gotowe – aplikacja jest przenośna i nie wymaga instalacji

---

## 🚀 Jak użyć

1. Uruchom **MonsteraSteamTool**
2. Kliknij **Baza Monstery**
3. Wpisz nazwę gry lub GameID
4. Kliknij **Dodaj**
5. Zrestartuj Steama

> **Optymalny czas testowania:** maksymalnie 2 godziny.  
> Wspieraj twórców i kupuj legalne kopie gier!
